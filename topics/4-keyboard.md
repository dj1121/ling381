# Don’t Touch The Keyboard Yet
Have you ever done a programming project? If so, you might know how useful it is
to write out a plan of your program on paper before you start coding. In this way,
you remember what each part of your program does and how it fits into the whole purpose
of the program. With academic writing, the idea is actually quite similar.
In this week, we will practice making a plan for your writing.

## Main Ideas
- PPPPPPP (The 7 Ps) 
- “Algorithmic” writing 
- Outlining and flow
- Remember the argument you’re making
